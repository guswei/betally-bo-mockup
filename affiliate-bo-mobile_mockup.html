RD 開發註記 — Dashboard

  • 格線桌面 4 欄統計卡 → 手機 1 欄（≥600px 可 2 欄）。
  • 金額DECIMAL(18,2)、右對齊、千分位；負值紅字。前後端都不可用 float 存金額。
  • i18n卡片標題為語系 key（已見印尼/英文），勿寫死字串。
  • TabMember Direct / Direct+Downline 手機改全寬 segmented。

RD 開發註記 — Player List

  • 寬表橫向捲動＋凍結首欄：position:sticky;left:0 並補背景色避免透字。
  • 凍結保持原欄序，凍結 Date of Registration（已拍板）。
  • 篩選單欄全寬、可收合面板；date range 手機全螢幕 picker。
  • 分頁縮為「上一頁／Page x of N／下一頁」；Items per page 用原生 select。
  • 狀態Status 用 chip（enum：ACTIVE / INACTIVE / …，以後端為準）。

RD 開發註記 — Commission Report

  • 超寬15+ 欄，橫向捲動＋凍結首欄（Year）。
  • ColumnsColumns Settings 手機保留；預設只開識別＋1~2 關鍵欄降低橫滑。
  • 數值金額欄 DECIMAL(18,2) 右對齊；Commission(%) 保留原精度。
  • Export右上下載維持於 app bar。

RD 開發註記 — Downline Commission

  • 兩表Upline / Downline 各自獨立橫滑容器，勿共用捲軸；區塊標題建議 sticky。
  • 篩選Game Vendor Channel / Game Type / date range 單欄堆疊。
  • PTPT Total / Comm 為百分比與金額並存，型別與精度依後端；Total Commission 可為負（見 -1,720.68），負值紅字。

RD 開發註記 — Player's Report

  • 篩選8 個欄位單欄堆疊；預設收合只露 Username＋日期。
  • 寬表13 欄，橫向捲動＋凍結首欄（Username）。
  • 金額Deposit/Withdraw/Rebate/Bonus/Revenue 皆 DECIMAL(18,2) 右對齊。
  • TabDirect / Direct+Downline 全寬 segmented。

RD 開發註記 — Tracking Link

  • 版面卡片堆疊全寬；長 URL word-break:break-all。
  • COPY按鈕 ≥44px、置於 URL 下方全寬；成功給英文 toast「Copied」。
  • 空狀態Exclusive Domain 給英文 empty-state，勿留空白。
